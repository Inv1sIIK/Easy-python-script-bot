A very easy Python script for a Telegram bot. If you don't want to bother, just copy and paste.
The script is made for people who don't understand how to do this and what is needed for this (basic understanding of Python)
The libraries you need to download are aiogram and aiohttp. You'll also need Python 3.10 or higher.
For the script to work properly, you'll need archives like red.zip and blue.zip in the script folder, as the script is for a bot that provides resource packs for Minecraft.
